# QuickRollingNotice
滚动公告、广告，支持灵活自定义cell。


##  安装 / Installation

方法一：`QuickRollingNotice` is available through CocoaPods. To install it, simply add the following line to your Podfile:

```
pod 'QuickRollingNotice'
```
